# Trained Models Storage

## Overview

Due to file size limitations imposed by GitHub, the trained autoencoder models produced in this project are **not stored directly in the repository**. 

To ensure **reproducibility and accessibility**, all trained models are stored externally using Google Drive.

## Usage Notes

- Models are saved in formats compatible with **TensorFlow / Keras**.
- To load a model, the corresponding architecture definition must be available.

## Reproducibility Statement 
External storage of model files is used due to size constraints and does not affect the user. The same models that are seen in the notebooks can be found here along the data used.
